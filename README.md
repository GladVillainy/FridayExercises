# FridayExercises

This repository contains learning exercises from school coursework during the 2nd semester.

The exercises are organized as weekly Friday assignments and focus on practicing programming concepts, problem-solving, and code structure.

## Structure
Each folder represents an individual Friday assignment with its own source code and, where relevant, local documentation.

## Updates
The repository is updated regularly, typically every weekend, as new exercises are completed.

## Purpose
The purpose of this repository is learning, practice, and documentation of progress throughout the semester.
